== Description ==
The plugin

Using this plugin you can create feedback form. Paste shortcode into article or page and create feedback form. Form has simple capcha for protect your email. Also, form has some options for configuration your mail in cpanel.

Используя данный плагин, вы сможете создать форму обратной связи на любой странице вашего сайта. Вставив соответствующий шорткод в текст страницы или статьи. А через панель администратора, можно настроить почтовый адрес для отправки сообщения, шаблон самого сообщения и прочие параметры.



== Installation ==
* Use standart WordPress instalator
* Or extract archive into "/wp-content/plugins/" folder
